# NVD Bundled Fonts

Inventory bundles regular Latin WOFF2 faces for the NVD editor. The files were downloaded from the official Google Fonts CSS API:

`https://fonts.googleapis.com/css2?family=<family>&display=swap`

The initial bundle contains only the normal 400 face used by the current editor. Additional weights, styles, and script subsets should be added when the editor can actually use them.

Family-specific license files from the public `google/fonts` repository are stored beside this manifest where available.

Google Sans and Google Sans Flex are served by the official Google Fonts API. Their official Google Fonts metadata reports `license: ofl` and `isOpenSource: true`:

- `https://fonts.google.com/metadata/fonts/Google%20Sans`
- `https://fonts.google.com/metadata/fonts/Google%20Sans%20Flex`

The applicable SIL Open Font License 1.1 template is stored as `SIL-OFL-1.1-template.txt`. The font files retain their embedded authorship and licensing metadata.
